# MEC E 692 – Assignment 1

**Author:** Seyed Amin Chabok  

## Description
This repository contains the Python code developed for **Assignment 1** of **MEC E 692**.  
The code generates a high-resolution plot of the function

\[
\ln(x^4) - 0.7
\]

with:
- Axes centered at \(x=0\) and \(y=0\)
- Clearly labeled tick marks on the axes
- The positive real root highlighted on the plot
- High-quality formatting suitable for academic reports

## Files
- `plot_ln_x4_minus_0p7.py` – Python script for plotting the function

## Requirements
- Python 3.x
- NumPy
- Matplotlib

## Usage
Run the script using:
```bash
python plot_ln_x4_minus_0p7.py
